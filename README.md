# Latra
**Latra** is a webtool that converts characters into Latin letters.

## Supported Languages
1. Russian
   - ISO 9
   - GOST
   - Mylstyle A & B
2. Ukrainian
   - ISO 9
   - Mylstyle A & B
3. Greek
   - Greek (Modern)
4. Georgian
   - Georgian
